# 最大爆仓显示说明

## 📊 显示内容

**最大爆仓**显示格式：
```
最大爆仓: 1.5000 / $95,000.00 (多头 🔴)
```

### 包含信息：
1. **数量**：1.5000 币（最大爆仓的币数量）
2. **金额**：$95,000.00（该笔爆仓的 USDT 金额）
3. **方向**：
   - **多头 🔴**（红色）- 表示这是多头爆仓（市场下跌导致）
   - **空头 🟢**（绿色）- 表示这是空头爆仓（市场上涨导致）

## 🎯 判断逻辑

- 统计时间窗口内（如 15 分钟）
- 找出**金额最大**的那一笔爆仓
- 显示该笔爆仓的数量、金额和方向

## 💡 市场意义

### 多头爆仓（红色）
- 表示大额多头仓位被强制平仓
- 说明市场**下跌压力大**
- 可能继续下跌或反弹

### 空头爆仓（绿色）
- 表示大额空头仓位被强制平仓
- 说明市场**上涨压力大**
- 可能继续上涨或回调

## ✨ 优化点

**修复前**：
```
最大爆仓: 1.5000 / $95,000.00  ❌ 不知道是多头还是空头
```

**修复后**：
```
最大爆仓: 1.5000 / $95,000.00 (多头 🔴)  ✅ 清晰标识方向
```

### 颜色区分
- 多头爆仓：**深红色** (DarkRed)
- 空头爆仓：**深绿色** (DarkGreen)

这样可以一眼看出最大爆仓是哪个方向，帮助判断市场压力！🚀
